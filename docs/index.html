Open-source VPS sustainability tool

VPS Energy Monitor

Estima el consumo energetico y las emisiones CO2e de un VPS Ubuntu con metricas locales de CPU, RAM y una metodologia transparente.

Que hace

Un recolector ligero para VPS Linux

Monitorea CPU y RAM Guarda muestras locales en SQLite usando datos de /proc.
Corre en background Incluye una unidad systemd para iniciar con Ubuntu.
Genera reportes Entrega estimaciones por dia, semana o mes desde la terminal.
Calcula CO2e Convierte energia estimada con un factor electrico regional.
Formula

Modelo de calculo

Para un VPS sin GPU, la estimacion usa potencia de CPU, RAM y PUE.

Energy kWh =
hours x ((vCPU x W_per_vCPU x CPU_usage)
+ (RAM_GB x W_per_GB_RAM)) / 1000 x PUE

CO2e =
Energy kWh x grid_emission_factor
Configuracion

Valores que debes ajustar

W_PER_VCPU Watts estimados por vCPU al 100% de uso.
W_PER_GB_RAM Watts estimados por cada GB de memoria RAM.
PUE Eficiencia energetica del centro de datos.
KGCO2E_PER_KWH Factor de emision electrica de la region del data center.
Instalacion

Uso rapido

1

Clona el repositorio

git clone https://github.com/garcelomarcia/vps-energy-monitor.git
cd vps-energy-monitor
2

Ejecuta el instalador

sudo ./install.sh
3

Edita la configuracion

sudo nano /etc/vps-energy.conf
4

Consulta reportes

vps-energy status
vps-energy report --period day
vps-energy report --period week
vps-energy report --period month
Limitacion

Estimacion, no medidor fisico

En un VPS, el hardware real se comparte con otros usuarios. Por eso el resultado debe entenderse como una estimacion basada en recursos virtuales y uso observado.

Para reportes formales, usa datos oficiales del proveedor cloud cuando esten disponibles.
